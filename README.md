# 주제

- STT(speech to text) 기능을 지원하는 실시간 화상, 음성 채팅 서비스

  

# 진행 과정

- [**1주차**](./1주차.md) : 기획

- [**2주차**](./2주차.md) : 실시간 텍스트 채팅 서비스 개발

- **3주차** : 실시간 음성(+화상) 채팅 서비스 개발

- **4주차** : 음성 채팅 내용을 텍스트로 변환하여 채팅으로 전송 개발

  

*  클릭하면 주차별로 작성한 md을 확인할 수 있습니다.
